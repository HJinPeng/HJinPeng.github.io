最佳实践

发表于 2018-05-02
1. 平稳退化

  1. 在html中 <a href="网址">
    在没有javascript的时候，可以点击跳转页面，返回需要按返回键

  2. 在.js中 onclick="popUp(this.getAttribute('href'))";return false;
    点击事件，return false;是禁止默认的窗口打开

2. 分离JavaScript
  1. window.onload
    在html全部加载完毕后会触发onload事件
  2. 加载一个函数，不需要（）。
3. 向后兼容

if(!getElementById) { return false; }
有些浏览器不支持使用DOM，或者被用户禁用了，所以要检测是否能识别该DOM函数，不能则返回false

4. 性能考虑
尽量少访问DOM和尽量减少标记

原：访问了两次DOM

改：访问了一次DOM

合并和放置脚本
  1. 只用一个 .js 文件。
  2. 把所有<script>标签放在文档末尾，</body>之前。
压缩脚本
  1. js和css：http://tool.oschina.net/jscompress/
