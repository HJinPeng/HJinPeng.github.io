  1. 向 div 元素添加圆角：

    div
{
border:2px solid;
border-radius:25px;
-moz-border-radius:25px; /* Old Firefox */
}

  2. 向 div 元素添加方框阴影：

    div
{
box-shadow: 10px 10px 5px #888888;
}

  3. 使用图片创建围绕 div 元素的边框：

    div
{
border-image:url(border.png) 30 30 round;
-moz-border-image:url(border.png) 30 30 round; /* 老的 Firefox */
-webkit-border-image:url(border.png) 30 30 round; /* Safari 和 Chrome */
-o-border-image:url(border.png) 30 30 round; /* Opera */
}

  4. 调整背景图片的大小：

    div
{
background:url(bg_flower.gif);
-moz-background-size:63px 100px; /* 老版本的 Firefox */
background-size:63px 100px;
background-repeat:no-repeat;
}

  5. 对背景图片进行拉伸，使其完成填充内容区域：

    div
{
background:url(bg_flower.gif);
-moz-background-size:40% 100%; /* 宽  高  老版本的 Firefox */
background-size:40% 100%;
background-repeat:no-repeat;
}

  6. 在 content-box 中定位背景图片：

    div
{
background:url(bg_flower.gif);
background-repeat:no-repeat;
background-size:100% 100%;
-webkit-background-origin:content-box; /* Safari */
background-origin:content-box; /*背景图片可以放置于 content-box、padding-box 或 border-box 区域。*/
}

  7. 为 body 元素设置两幅背景图片：

    body
{ 
background-image:url(bg_flower.gif),url(bg_flower_2.gif);
}

  8. 向标题添加阴影：

    h1
{
text-shadow: 5px 5px 5px #FF0000;
}

  9. @font-face

    @font-face
{
font-family: myFirstFont;
src: url('Sansation_Light.ttf'),
    url('Sansation_Light.eot'); /* IE9+ */
}

div
{
font-family:myFirstFont;
}

  10. translate() 方法
    通过 translate() 方法，元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数：

    div
{
transform: translate(50px,100px);
-ms-transform: translate(50px,100px);        /* IE 9 */
-webkit-transform: translate(50px,100px);    /* Safari and Chrome */
-o-transform: translate(50px,100px);        /* Opera */
-moz-transform: translate(50px,100px);        /* Firefox */
}

  11. rotate() 方法
    通过 rotate() 方法，元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转。

    div
{
transform: rotate(30deg);
-ms-transform: rotate(30deg);        /* IE 9 */
-webkit-transform: rotate(30deg);    /* Safari and Chrome */
-o-transform: rotate(30deg);        /* Opera */
-moz-transform: rotate(30deg);        /* Firefox */
}

  12. scale() 方法
    通过 scale() 方法，元素的尺寸会增加或减少，根据给定的宽度（X 轴）和高度（Y 轴）参数：

    div
{
transform: scale(2,4);
-ms-transform: scale(2,4);    /* IE 9 */
-webkit-transform: scale(2,4);    /* Safari 和 Chrome */
-o-transform: scale(2,4);    /* Opera */
-moz-transform: scale(2,4);    /* Firefox */
}

  13. skew() 方法
    通过 skew() 方法，元素翻转给定的角度，根据给定的水平线（X 轴）和垂直线（Y 轴）参数：

    div
{
transform: skew(30deg,20deg);
-ms-transform: skew(30deg,20deg);    /* IE 9 */
-webkit-transform: skew(30deg,20deg);    /* Safari and Chrome */
-o-transform: skew(30deg,20deg);    /* Opera */
-moz-transform: skew(30deg,20deg);    /* Firefox */
}

  14. 过渡

动画基础知识

  1. 位置
    1. position属性的合法值：static,fixed,relative,absolute
    2. 方位，避免冲突：top/bottom,left/right

  2. 时间

    1. movement=setTimeout("function",interval);
      第一个参数：通常是“字符串”
      第二个参数：数值，以毫秒为单位设定了需要经过多长时间才开始执行第一个参数给的函数
      movement: 全局变量，可在其他地方被取消动作clearTimeout(movement);

    2. 抽象

    对这句括号中的理解：
    movement=setTimeout("moveMessage('"+id+"',"+xx+","+yy+","+zz+")",zz);
    第一个参数是 字符串，括号中的参数也是由字符串组成，”moveMessage(“ 这是第一个，id是第二个，”‘,”是第三个，这三个就组成了moveMessage('id',，其中id是字符串，没有单引号会被解析成变量。

    使用：
    moveMessage("message",200,100,10);

实用动画

  1. 如果把position:absolute的元素A放入一个position:relative的元素B中，B就成为A的容器元素，A将在B的显示区域里按absolute方式进行摆放

  2. 创建属性：element.property=value
    比如创建一个名为foo的属性，值为bar：element.foo="bar"

  3. movement全局变量的停止处理：

把movement变成elem的属性，查询它是否存在，如果在moveElement函数执行时已经有一个movement属性，就用clearTimeout对它进行复位。

使用到的一些函数

1.parseInt("字符串") parseFloat("字符串")把字符串中的数值信息提取出来
2.Math.ceil(number)大于number的最小整数
Math.floor(number)小于number的最大整数
Math.round(number)最近number的整数

  1. 开启/关闭mysql
    net start mysql
    net stop mysql

  2. 进入/退出mysql
    mysql -uroot -pmysql
    exit

  3. 输入错误，回到上级
    \c

  4. 创建数据库
    CREATE DATABASE

  5. 修改数据库
    ALTER DATABASE t2 CHARACTER SET = utf8;

  6. 显示所有数据库
    SHOW DATABASES

  7. 显示某个数据库
    SHOW CREATE DATABASE t2

  8. 删除数据库
    DROP DATABASE t1;

  9. 使用数据库
    USE 数据库名

  10. 创建表

    CREATE TABLE tb1(
-> username VARCHAR(20),
-> age TINYINT UNSIGNED,
-> salary FLOAT(8,2) UNSIGNED
-> );

  11. 查看表
    SHOW TABLES;
    SHOW TABLES FROM mysql;

  12. 查看数据表结构
    SHOW COLUMNS FROM tb1;

  13. 记录的插入
    INSERT 表名 VALUES(全部数据)
    INSERT 表名(行名) VALUES(部分数据)
    INSERT users VALUES(NULL,'JOHN',123,2,1); NULL默认自动编号
    INSERT users VALUES(DEFAULT,'TOM',123,2,1); DEFAULT==NULL;
    INSERT users SET username='BEN',password='456';

  14. 记录的查询
    SELECT * FROM 表名

  15. 数据不能为空

     CREATE TABLE tb2(
-> username VARCHAR(20) NOT NULL,
-> age TINYINT UNSIGNED NULL
-> );

  16. 主键PRIMARY KEY

    1. 自动编号＋主键

      CREATE TABLE tb3(
-> id SMALLINT UNSIGNED AUTO_INCREMENT     PRIMARY KEY,
-> username VARCHAR(30) NOT NULL
-> );

    2. 主键（无自动编号，不可重复数据）

      CREATE TABLE tb4(
-> id SMALLINT UNSIGNED PRIMARY KEY,
-> username VARCHAR(30) NOT NULL
-> );

  17. 唯一约束UNIQUE KEY
    username VARCHAR(20) NOT NULL UNIQUE KEY,

  18. 默认值DEFAULT
    sex ENUM('1','2','3') DEFAULT '3'

  19. 外键约束 FOREIGN KEY(子表的列名) REFERENCES 父表 (父表的列名)

    CREATE TABLE provinces(
-> id SMALLINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
-> pname VARCHAR(20) NOT NULL
-> );

CREATE TABLE user(
-> id SMALLINT UNSIGNED PRIMARY KEY AUTO_INCREMENT,
-> username VARCHAR(10) NOT NULL,
-> pid SMALLINT UNSIGNED,
-> FOREIGN KEY (pid) REFERENCES provinces (id)
-> );

  20. 添加删除列
    ALTER TABLE users1 ADD password VARCHAR(30) NOT NULL AFTER username;
    ALTER TABLE users1 DROP truename;

  21. 添加约束
    主键约束：ALTER TABLE users2 ADD CONSTRAINT PK_users2_id PRIMARY KEY (id);
    唯一约束：ALTER TABLE users2 ADD UNIQUE (username);
    外键约束：ALTER TABLE users2 ADD FOREIGN KEY (pid) REFERENCES provinces (id);
    默认约束：ALTER TABLE users2 ALTER age SET DEFAULT 15;
    删除默认约束：ALTER TABLE users2 ALTER age DROP DEFAULT;
    删除唯一约束：

    查看约束名：`SHOW INDEXES FROM users2\G;`
删除索引：`ALTER TABLE users2 DROP INDEX username;`

    删除外键约束 ：

    查看外键约束名：`SHOW CREATE TABLE   users2;`
如CONSTRAINT `users2_ibfk_1`
删除：ALTER TABLE users2 DROP FOREIGN KEY users2_ibfk_1;

  22. 查看约束名字
    普通形式：SHOW INDEXES FROM users2;
    网格形式：SHOW INDEXES FROM users2\G;

  23. 修改列定义 MODIFY
    ALTER TABLE users2 MODIFY id TINYINT UNSIGNED NOT NULL FIRST;

  24. 修改列名称（同时修改列定义）
    ALTER TABLE users2 CHANGE pid p_id TINYINT UNSIGNED NOT NULL;

  25. 修改数据表的名称
    ALTER TABLE users2 RENAME users3;
    RENAME TABLE users3 TO users2;

  26. 单表更新
    UPDATE users SET age=age-id,sex=0;
    UPDATE users SET age=age+10 WHERE id%2=0;

  27. 单表删除
    DELETE FROM users WHERE id=6;
    下一个插入的数据id不会是删除的这个。

  28. 查询数据
    SELECT id,username FROM users; 先id再username
    SELECT username,id FROM users; 先username再id
    SELECT users.id,users.username FROM users;某个数据表的列名用.

  29. 别名AS
    SELECT id AS userID,username AS uname FROM users;改成AS后的名字

  30. GROUP BY 语句对查询结果分组
    SELECT sex,age FROM users GROUP BY sex,age;

    +------+-----+
| sex  | age |
+------+-----+
| NULL |  10 |
|    0 |   4 |
|    0 |   6 |
|    0 |  15 |
|    0 |  21 |
|    1 |   2 |
+------+-----+

    SELECT sex FROM users GROUP BY sex HAVING count(id)>3;

  31. 对查询结果进行排序[ASC|DESC]
    SELECT * FROM users ORDER BY age DESC,id DESC;

  32. 限制查询数量
    SELECT * FROM users LIMIT 2,2; 从0开始算的

  33. 添加数据到另一个表
    INSERT test(username) SELECT username FROM users WHERE age>=10;
